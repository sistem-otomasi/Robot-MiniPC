## ROS 2 Commands Berguna

```bash
# List semua packages
ros2 pkg list

# Info package
ros2 pkg prefix robot_example

# List executables dalam package
ros2 run robot_example --help

# Build hanya package tertentu
colcon build --packages-select robot_example

# Build dengan symlink (development mode)
colcon build --symlink-install

# Clean build artifacts
colcon clean
```

## Key Concepts

### Ament (ROS 2 Build System)
- **package.xml**: Metadata, dependencies, build type
- **setup.py/setup.cfg**: Python package configuration
- **buildtool_depend**: ament_python untuk Python packages

### Colcon (Build Tool)
- Menggunakan plugin untuk berbagai build system (ament_cmake, ament_python, dll)
- Parallel build capabilities
- Automatic dependency resolution

### ROS 2 Python Nodes
- Inherit dari `rclpy.node.Node`
- Menggunakan `create_publisher()` dan `create_subscription()`
- Message types dari std_msgs, sensor_msgs, geometry_msgs, dll
